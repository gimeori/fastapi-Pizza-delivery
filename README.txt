POST /api/order/  Создание заказа
PUT /api/order/status/{order_id}/ Обновление статуса заказа
DELETE /api/order/delete/{order_id}/ Удаление заказа
GET /api/orders/ Список всех сделанных заказов
GET /api/orders/{order_id}/ Информация об одном заказе



POST /api/item/  Добавление пиццы
GET /api/items/ Список всех пицц
GET /api/items/{item_id}/ Выбор одной пиццы
DELETE /api/item/delete/{item_id}/ Удаление пиццы
PUT /api/items/update/{item_id}/ Обновление информации о пицце


GET /docs/ Документация API
